Node JS

Ultima provocare pentru un dezvoltator incepator este node js. Node js este JavaScript pe server.

Avantajele folosirii node sunt:

  • Acelasi limbaj de programare si pe client si pe server, cod re-utilizabil la maxim.
  • Viteza buna
  • Sistemul asincron puternic din JavaScript
  • Comunicare simpla cu clientii prin websockets, Ajax

Dezvantajele folosirii node sunt:

  1. Usor de invatat, greu de stapanit
  2. Sistemul asincron este greu de stapanit
  3. Creearea unui server pe socket poate fi dificila
